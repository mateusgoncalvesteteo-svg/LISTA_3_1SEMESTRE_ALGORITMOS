Para i de 0 até tamanho(texto)-1 faça
    Se texto[i] for maiúscula então
        texto[i] ← ((texto[i] - 'A' + N) mod 26) + 'A'
    Senão se texto[i] for minúscula então
        texto[i] ← ((texto[i] - 'a' + N) mod 26) + 'a'
FimPara
